# 🏦 Loan Approval AI System

An AI-powered Loan Approval Prediction web application built using Streamlit and Machine Learning.

## 🚀 Features

* Loan approval prediction
* Random Forest Classifier
* Interactive Streamlit dashboard
* ROC Curve & Confusion Matrix
* Feature Importance Analysis
* Real-time prediction system

## 📊 Technologies Used

* Python
* Streamlit
* Scikit-learn
* Pandas
* NumPy
* Matplotlib
* Seaborn

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🌐 Deployment

Deploy easily on Streamlit Cloud using GitHub integration.
